📦 Product Matcher App
│
├── 1. Web Scraping Module
│   └── ดึงข้อมูลจากเว็บไซต์คู่แข่ง (ชื่อ, รูปภาพ, ราคา)
│
├── 2. Database Module
│   └── จัดเก็บข้อมูลสินค้าในฐานข้อมูล (เช่น SQLite หรือ Pandas DataFrame)
│
├── 3. Input Module
│   ├── รับชื่อสินค้าจากผู้ใช้
│   └── หรือ รับรูปภาพสินค้าจากผู้ใช้
│
├── 4. Matching Engine
│   ├── ถ้าเป็นชื่อ: ใช้ NLP (เช่น fuzzy matching หรือ embedding)
│   └── ถ้าเป็นรูปภาพ: ใช้ image embedding (เช่น CLIP) + cosine similarity
│
└── 5. Streamlit UI
    ├── ปุ่มดึงข้อมูลจากเว็บคู่แข่ง
    ├── ช่องกรอกชื่อ / อัปโหลดรูป
    └── แสดงผลสินค้าที่คล้ายกัน (ชื่อ, ราคา, รูปภาพ)


🔄 Flow การทำงานของผู้ใช้
ผู้ใช้เปิดแอป → เห็นปุ่ม “ดึงข้อมูลจากคู่แข่ง”
เมื่อกดปุ่ม → ระบบดึงข้อมูลจากเว็บคู่แข่งและเก็บไว้ในฐานข้อมูล
ผู้ใช้กรอกชื่อสินค้า หรือ อัปโหลดรูปภาพ
ระบบจับคู่สินค้าอัตโนมัติจากฐานข้อมูล
แสดงผลสินค้าที่คล้ายกัน พร้อมชื่อ, ราคา, และรูปภาพ
